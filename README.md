# 基于vue2+vue-router+webpack+less+es6仿美团外卖项目

> A Vue.js project

## Build Setup


>install dependencies

```
$ yarn
```
>serve with hot reload at localhost:8080

```
$ npm run dev
```

>build for production with minification

```
npm run build
```
